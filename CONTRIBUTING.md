# 🤝 Contribuir a ClaroScan

¡Gracias por interesarte en mejorar **ClaroScan**! Queremos que esta sea la herramienta de escaneo más limpia y privada de Android.

## 🚀 ¿Cómo puedes ayudar?

Puedes contribuir de varias formas:

1.  **Reportar Errores:** Si el filtro de fondo blanco falla en ciertas condiciones de luz, abre un "Issue".
2.  **Sugerir Funciones:** ¿Quieres que la app reconozca texto (OCR)? Cuéntanos tu idea.
3.  **Mejorar el Código:** * Haz un "Fork" del proyecto.
    * Crea una rama para tu mejora (`git checkout -b mejora-filtro`).
    * Envía un "Pull Request".

## 🛠️ Guía de Estilo

Para mantener la esencia de **ClaroScan**, pedimos que:
* **Nombres Limpios:** No uses prefijos raros ni números genéricos.
* **Privacidad Primero:** No añadas librerías que envíen datos a servidores externos.
* **Código Legible:** Usa Kotlin moderno y mantén el diseño minimalista.

## ⚖️ Al contribuir
Al enviar código a este proyecto, aceptas que tus contribuciones estarán bajo la **Licencia MIT**.
