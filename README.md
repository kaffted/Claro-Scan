# 📄 ClaroScan: Escaneo Impecable y Privado ✨

**ClaroScan** es una aplicación de Android de código abierto diseñada para digitalizar documentos con la máxima claridad. Sin rastreadores, sin suscripciones y sin procesos en la nube. 

Lo que ves es lo que obtienes: un documento **limpio, con fondo blanco puro y texto nítido.**

---

## ✨ Características Principales

* **Fondo Blanco Inteligente:** Algoritmo adaptativo que elimina sombras y manchas grises del papel.
* **Nomenclatura Automática Intuitiva:** Los archivos se guardan siguiendo el formato: `scan_día_mes_año_hora_periodo` (Ej: `scan_18_03_2026_01_13_pm`).
* **Privacidad por Diseño:** Procesamiento 100% local. Tus documentos nunca salen de tu dispositivo.
* **Interfaz Minimalista:** Sin publicidad, sin menús confusos y sin cadenas de texto innecesarias (`+1`, `+2`, etc.).
* **Exportación PDF de Alta Calidad:** Genera documentos ligeros y listos para imprimir o compartir.

---

## 🚀 Cómo Funciona

1.  **Captura:** Abre la cámara y encuadra tu documento.
2.  **Mejora:** La app aplica un filtro de umbral adaptativo para blanquear el fondo automáticamente.
3.  **Guarda:** El PDF se genera al instante con el nombre cronológico exacto.
4.  **Gestiona:** Visualiza tus escaneos en una lista limpia y organizada.

---

## 🛠️ Detalles Técnicos

* **Lenguaje:** Kotlin 
* **Procesamiento de Imagen:** OpenCV (Filtro Gaussiano + Adaptive Thresholding)
* **Motor PDF:** Android Native PdfDocument
* **Diseño:** Material Design 3 (Material You)
